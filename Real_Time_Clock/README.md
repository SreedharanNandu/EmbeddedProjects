**REAL TIME CLOCK**

Note:- The source code published here is only for hobby/school/university study purpose and shall not be used for any commercial product/license.

This project uses Microchip PIC12 MCU to tick time.
MCU :- PCI12F1840
RTC Chip : - ISL1208.
